1. "a" = contains a hyperlink, "p" defines a paragraph
  2. a element p element
  3. "a" is a child of "p"
  4. first one is the opening tag, second one is the closing tag
  5. href, id, class, target
  6. href value="https://google.com", id value = "the-best-link", class value = "banana", target value = "_blank"
  7. href is the hyperlink, its value is google.com. it has an id of "the-best-link" which you can use to apply CSS rules to (will ONLY apply to this element). it has the class "banana", so any CSS you give to the class "banana" will apply to this element (unless it is overwritten by something in "the-best-link"). target = "_blank" means the target link will open in a blank window.
